Vigenere

Problem Set 2. For detailed info visit the CS50x course page.



Rather than cracking codes, this time we're making codes, using Vigenere's cipher.



Upon supplying a keyword, which will be used to encrypt a message (a.k.a. "plaintext"), and, of course, the message, the program encrypts the message with the keyword. Each letter in the keyword equates to a numeric value (a=0, b=1, c=2, etc). Each letter in the plaintext is taken in turn and encrypted (converted to "ciphertext") by adding the value of the corresponding keyword letter to it.


For example, using a keyword of "d" (=3) and plaintext "p", the resulting ciphertext will be "s". If the plaintext has more letters than the keyword you simply loop back to the start of the keyword.

The first challenge in CS50x.

A game of tennis, programmed in Scratch.


Treat yourself to a game at the Scratch website, if only for the soundtrack! In Scratch, you can see the code by clicking on the blue "See Inside" button at the top right of the page.


Scratch is a great way to get into programming because its very visual and is not imposing like common languages such as C or Java might be, whilst still teaching some programming fundamentals like loops, variables and conditionals. The website features some great examples of what can be achieved with Scratch; take a look and explore the possibilities!
