Students are able to choose their own subject for the final piece of work. I chose to program in C because I had enjoyed an earlier project ("Speller" - see below) and I had an idea to implement an anagram-creating program. Try it out yourself in the embedded window below.

Mashup

Problem Set 8. Go here for the CS50x course page.

The challenge here is to combine Google Maps with RSS news feeds in order to create a web-based, searchable map which yields local news items when a location is selected by the user. There is also a search box which references a database of placenames to provide suggestions as the user types.


Although only four functions (two in Python; two in JavaScript) need to be written, a good understanding of how the JavaScript, Ajax, Google APIs, SQLite, typeahead.js, XML, jQuery, Flask, and Python parts work together is needed in order to write the code for the four functions.


Watch this video to get a better flavour!


View or fork the code at my GitHub account.

C$50 Finance

Problem Set 7. Go here for the CS50x course page.

This project implements a fake share-trading website. Most of the code is provided in the course materials, but as with 'Mashup', a solid understanding of what's going on in the entire code-base (Python, Flask, SQLite, JSON, HTML) is needed in order to complete the project.

My Python code consists of six functions that are typical of e-commerce sites, such as registering a new user, buying and selling, looking up prices on a genuine financial-services website, and providing a history of all past transactions.

Sentiments

Problem Set 6. Go here for the CS50x course page.

Time to get intimate with Twitter's APIs. Sentiments requires three separate programs to be written...

  • Smile requires Python code that returns a different output for any given word, depending on whether the word's sentiment is good, bad or neutral.
  • Tweets requires Python code to return a Twitter user's recent tweets; each one coloured red, yellow or green depending on whether the overall sentiment of the tweet is goo, bad or neutral.
  • Sentiments Pie requires Python code to display a pie chart on a local web server. The pie is split into red, yellow and green portions, depending on the overall sentiments of a Twitter user's recent tweets.

Speller

Problem Set 5. Go here for the CS50x course page.

A project to complete the implementation of a spellchecker, in C. The requirement is to use some kind of search tree (e.g. linked lists, hash tables, or tries) to create a dictionary which will then be used to check the spelling of words from texts, ranging from the King James Bible to Dickens to Ralph Wiggum.


View or fork the code at my GitHub account.


My weapon of choice was a trie. This project was both more challenging and more rewarding that earlier projects, primarily due to the memory-management that is required with C.
