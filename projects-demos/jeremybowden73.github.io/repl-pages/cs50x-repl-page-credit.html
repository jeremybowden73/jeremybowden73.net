Credit demo

The top half of this embedded window is the code, so probably best not to touch that unless you know what you're doing. If it goes pear-shaped just reload the page (Ctrl-R or Cmd-R).


Click the "Play" triangle (bottom right corner).

You'll need to enter a genuine Visa, Mastercard (only one starting with 5, per the requirements of the problem) or AmEx credit card number. If you do, the program responds by telling you which card you've got.


If you don't have one to hand, this Paypal developers site has some dummy numbers you could try. Ignore the MasterCard numbers that start with 2 though!
