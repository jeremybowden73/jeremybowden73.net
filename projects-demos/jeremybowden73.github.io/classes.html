Self-learning Classes/Courses

  • For me it started old-school, with a book: C Programming, Absolute Beginner's Guide by Greg Perry and Dean Miller.

    This book lays out the basics of C in small lessons and exercises, covering many fundamentals of programming languages (variables, loops, if...else, arrays, file access, functions, pointers).


  • After a little research into what would be good to move on to next, I enrolled for the online version of Harvard University's "Introduction to Computer Science" through edX. Head over to the CS50x course information page on edX for more detailed info on the course.

    After some more practice with C the course moves on to other elements in programming: Python, SQL, JavaScript, jQuery, APIs, frameworks, Ajax, JSON, HTML... I think that's all! The course is excellent: the course materials, the quality of the teaching, the amount of stuff that students learn, the community of learners and teachers that support it. To see the projects I worked on, click here.


  • I had a desire to get myself a website (a bit like this one!), but instead of going to Squarespace or Wix I wanted to build it myself, right from File>New, so I thought it might be a good idea to learn a little bit of HTML and CSS.

    There are over 3 gazillion front-end web developer courses out there, but I liked the look of the Mozilla Developer Network tutorials. I spent a while following the logically-ordered course materials and then - hey presto! - this website appeared! (The source code is available at my GitHub.)
