HTML


HTML is the "content" of a website; the words and pictures that the visitor to the website can read. It can also include some formatting, like the single black line break above. The MDN website covers all the need-to-know HTML stuff like this (plus a ton more!):

  • The format of elements (each individual piece of a website's content)
  • Element attributes
  • Lists
  • Hyperlinks
  • Formatting text, like italics or computer code
  • Debugging HTML
  • Responsive design (designing websites for all devices)
  • Embedding external content
  • Resolution switching; Art direction
  • Tables
  • Forms (and sending the data contained therein
  • Buttons; Checkboxes; Radio buttons
  • Drop-down content



CSS


CSS is the means by which the HTML is presented to the visitor to the website; the way it is styled and laid out on the screen. It's important to ensure that the website is not only functional but also looks it's best on any device: laptop, desktop, tablet, or mobile. This is known as Responsive Design.

  • Rules (formatting the HTML elements)
  • The DOM (Document Object Model; the 'roadmap' of a website)
  • Selectors; combinators
  • Values and units
  • Colours
  • Cascade and Inheritance
  • Box model; styling boxes
  • Styling text, tables, and lists
  • Backgrounds; background images
  • Positioning elements; floats
  • Flexible boxes and Grids
  • Responsive design
